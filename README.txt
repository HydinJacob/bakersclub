COCHIN BAKERS

Its a simple model of a responsive web application for one of the bakers club in our location. 
The landing page includes sections such as navigation, gallery, events and newsletter subscription.
It also provides the feature to join the events by a registration process, along with a form to input
some general information. Finally, if the customers likes to recieve the newsletters, we provide a 
facility for subscribing the same.  




Technologies used are  HTML, CSS and Javascript.

HTML and CSS are used for structuring and styling the web page. 
While, Javascript is used for retrieving data from the form.
